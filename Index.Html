1. **Sum of Array Elements:** Write a function that calculates the sum of all the elements in an array of numbers.
2. **Find Maximum Value:** Create a function that finds the maximum value in an array of numbers.
3. **Filter Even Numbers:** Write a function that takes an array of numbers and returns a new array with only the even numbers.
4. **Reverse an Array:** Create a function that reverses the order of elements in an array without modifying the original array.
5. **Remove Duplicates:** Write a function that removes duplicate elements from an array while preserving the original order.
6. **Concatenate Arrays:** How do you concatenate two arrays in JavaScript?
7. **Slice and Splice:** Explain the difference between `slice` and `splice` methods when working with arrays.
8. **Sort an Array:** How can you sort an array of strings in alphabetical order?
9. **Search for an Element:** Write a function that searches for a specific element in an array and returns its index, or -1 if it's not found.
10. **Array Manipulation:** Given an array of numbers, write a function that multiplies each element by a specified factor and returns the modified array.
11. **Find the Average:** Write a function that calculates the average (mean) of all the elements in an array of numbers.
12. **Remove All Occurrences:** Create a function that removes all occurrences of a specific value from an array and returns a new array without those values.
13. **Flatten Nested Arrays:** Write a function that flattens a nested array (an array of arrays) into a single flat array.
14. **Count Occurrences:** Create a function that counts how many times a specific value appears in an array.
15. **Unique Values:** Write a function that returns an array containing only the unique values from the original array (removing duplicates).
16. **Chunk into Groups:** Create a function that chunks an array into smaller arrays of a specified size. For example, chunk `[1, 2, 3, 4, 5]` into groups of size 2: `[[1, 2], [3, 4], [5]]`.
17. **Rotate Array:** Write a function that rotates the elements of an array to the right by a given number of positions. For example, rotating `[1, 2, 3, 4, 5]` by 2 positions would result in `[4, 5, 1, 2, 3]`.
18. **Merging Arrays:** Combine two arrays into a single array without any duplicate values.
19. **Sum of Two Arrays:** Create a function that takes two arrays of numbers and returns a new array containing the element-wise sum of the two arrays.
20. **Shuffle an Array:** Write a function that shuffles the elements of an array randomly so that the order is different each time you call the function.
